# Entorno virtual

## Crear
```bash
py -3 -m venv .venv
```

## Activar
```bash
.venv\Scripts\activate
```

## Desactivar
```bash
deactivate
```

# Instalar dependencias
```bash
pip install -r requirements.txt
```

# Variables de entorno
Copiar el archivo `env.template` y renombrarlo `.env`.

# Correr
```bash
uvicorn app.main:app --reload
```
